Definiciones de Términos Importantes

Aquí encontrarás definiciones de palabras clave utilizadas en el sitio web, enfocadas en soporte técnico para estudiantes de bachillerato en informática. Estas explicaciones te ayudarán a comprender mejor los conceptos.

Término Definición
Soporte Técnico Servicio que ayuda a resolver problemas relacionados con equipos informáticos, software y sistemas, asegurando su funcionamiento óptimo.
Antivirus Programa diseñado para detectar, prevenir y eliminar virus, malware y otras amenazas cibernéticas en una computadora.
Malware Software malicioso que incluye virus, troyanos y ransomware, diseñado para dañar o robar información de un sistema.
Procesador (CPU) Componente principal de una PC que ejecuta instrucciones y procesa datos, actuando como el "cerebro" del equipo.
Memoria RAM Almacenamiento temporal de datos que permite el acceso rápido a información activa, mejorando el rendimiento multitarea.
Disco Duro (HDD/SSD) Dispositivo de almacenamiento permanente para datos, archivos y programas. Los SSD son más rápidos que los HDD tradicionales.
Tarjeta Madre Placa principal que conecta todos los componentes de una PC, facilitando la comunicación entre ellos.
Fuente de Alimentación Componente que proporciona energía eléctrica estable a todos los elementos de la PC, convirtiendo corriente alterna en directa.
Mantenimiento Conjunto de acciones preventivas y correctivas para mantener una PC en buen estado, como limpieza y actualizaciones.
Diagnóstico Proceso sistemático para identificar y resolver fallas en una PC, siguiendo pasos como inspección y pruebas.
Firewall Barrera de seguridad que monitorea y controla el tráfico de red entrante y saliente para proteger contra accesos no autorizados.
Actualización Proceso de instalar nuevas versiones de software o drivers para corregir errores, mejorar seguridad y agregar funcionalidades.
Optimización Técnicas para mejorar el rendimiento de una PC, como eliminar archivos temporales o gestionar programas de inicio.
Inspección Visual Revisión física de componentes y conexiones para detectar daños o problemas evidentes en una PC.
